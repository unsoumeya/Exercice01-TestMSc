# Test d’admission MSc Cyber

Ce dépôt contient les exercices réalisés dans le cadre du test d’admission au MSc Cyber.

## Contenu

- **Exercice 01 – Packet Tracer**
  - Mise en place d’un mini-lab réseau avec VLAN, DHCP et routage

- **Exercice 02 – Active Directory**
  - Création d’un domaine Active Directory
  - Import automatisé des utilisateurs et groupes via PowerShell

- **Exercice 03 – Docker**
  - Déploiement d’un site WordPress avec Docker Compose

Chaque exercice est disponible dans son dossier dédié.
Les captures d’écran et les explications détaillées sont fournies dans le PDF de rendu final.
